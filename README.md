#Como iniciar
Instale o python-pygame
```bash
	sudo apt-get install python-pygame
```
Execute ***main.py*** com o python2:
Para rodar sem pygame:
```bash
	python main.py nomeMapa
```
Para rodar com pygame:
```bash
	python main.py -pygame nomeMapa
```
